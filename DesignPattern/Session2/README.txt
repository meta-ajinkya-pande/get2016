Create a simple web application using MVC2 Design Pattern (using the design pattern assignments) for the following pages -
Landing Page
Should have a short description about Metacube Software Pvt. Ltd.
Should have a link to List Page
Should have a link to Create Item page
List Page
Should list out all the cars and their details in a tabular format. Each list item will have a ‘Edit’ and ‘View’ link
Details Page
Should list out all the details of the item (Car)
Should have link to Landing Page
New Car Create Page / Car Edit Info Page (Common page should be used)
Should have a Create / Edit form with all the fields applicable for the Item (Car)
Upon successful registration/edit, should redirect to List Page
